# Metin2 Escrow – bezpieczny handel itemami

Platforma escrow dla graczy Metin2. Kupujący wpłaca pieniądze do depozytu, sprzedający wysyła item, a środki są zwalniane po potwierdzeniu odbioru.

## 🚀 Uruchomienie lokalne

1. Zainstaluj Node.js (v16+)
2. Sklonuj repozytorium
3. W terminalu w folderze projektu:
