# BrainUp – demo UI (oryginalne)
To jest w pełni oryginalny, statyczny szablon HTML/CSS/JS inspirowany układem z obrazka.
Nie używa zewnętrznych bibliotek ani zasobów.

## Uruchomienie
Otwórz plik `index.html` w przeglądarce.

## Struktura
- index.html – strona główna
- *.html – podstrony (działają linki)
- assets/style.css – styl
- assets/app.js – prosta logika filtrów tabeli
- assets/brainup-logo.svg – oryginalne logo SVG
